# RCQ Android

Android client for **RCQ** — privacy-first messenger with 9-digit IDs, no
phone, no email, end-to-end encrypted by default.

> **Status:** pre-alpha, in active development. Not yet shipped to the
> Play Store. Cross-platform message compatibility with the iOS client
> is being wired up via `libsignal-android`.

- Main site: <https://rcq.app>
- iOS client (reference implementation): <https://github.com/rcq-messenger/rcq-ios>
- Issue tracker: this repo's [Issues](../../issues)
